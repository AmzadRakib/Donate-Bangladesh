Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a structured representation of a web page. It allows JavaScript to interact with and change the HTML and CSS of a website. Think of it as a tree where every HTML element (like div, p, img) is a "node" that can be accessed, modified, or removed.

How do you select on element from the DOM?

To select an element from the DOM, use:
-document.getElementById('id') for an element with a specific ID.
-document.querySelector('selector') for the first match of a CSS selector (like #id, .class).
-document.querySelectorAll('selector') for all matching elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you add an event listener to a parent element to handle events on its child elements. It’s useful because:
- It reduces the number of event listeners.
- It works for dynamically added elements.
